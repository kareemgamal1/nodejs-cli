## ğŸ“¦ Commands

```bash
# Add a new note
notes new "Finish reading Atomic Habits" --tags=books,habits

# List all notes
notes all

# Search notes by keyword
notes find "atomic"

# Remove a note by ID
notes remove <id>

# Remove all notes
notes clean

# View notes in browser (default port: 5000)
notes web [port]
```
